Aide pour la saisie du Strum

Pour saisir un rythme, utilisez les lettres suivantes dans la boîte de saisie :

  • B / b : Coup vers le bas (Fort / Normal)
  • H / h : Coup vers le haut (Fort / Normal)
  • X / x : Chunk - coup étouffé (Fort / Faible)
  • E / e : Égréné / Arpégé (Fort / Doux)
  • Espace ou Tiret (-) : Silence / Pause

L'affichage sous la saisie se met à jour automatiquement pour vous montrer les mesures et les temps détectés.
